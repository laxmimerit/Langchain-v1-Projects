RecursiveCharacterTextSplitter: Split large documents into manageable chunks

Why? Pages are too coarse - smaller chunks prevent context "wash-out"

Overlap: Preserves context between chunks

  chunk_size=4096,
  chunk_overlap=410
)

üß† Embeddings

Vector Representation: Convert text to numeric vectors

Similar meanings = geometrically close vectors

Enables semantic search without exact keyword matching

üóÑÔ∏è Vector Storage

Chroma Database: Store and index embeddings for fast similarity search
