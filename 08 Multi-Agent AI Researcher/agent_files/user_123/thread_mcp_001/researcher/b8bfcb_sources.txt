METADATA:
- Core Participants: Host, Client, Server.
- Primary Wire Format: JSON-RPC 2.0.
- Transports: stdio (local), SSE (remote/HTTP).
- Key Primitives: Tools, Resources, Prompts, Sampling, Elicitation.
- Lifecycle: Initialize -> Operation -> Shutdown.
